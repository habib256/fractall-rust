# harness/ — mémoire de la boucle d'auto-amélioration

Scorecards versionnés : `history/<UTC>-<sha>.json` (mémoire longue),
`baseline.json` (référence figée via `harness.py baseline`). Le protocole et
les trois axes (vitesse / parité / qualité) sont décrits dans `../HARNESS.md` ;
l'outil est `../scripts/harness.py` ; le dernier score lisible est
`../SCORECARD.md`.
